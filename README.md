# awsprofile-cli

Interactive AWS profile switcher using `fzf`.

## Installation
```bash
#bash
git clone https://github.com/marques576/awsprofile-cli.git && echo 'source "$PWD/awsprofile-cli/awsprofile.sh"' >> ~/.bashrc

#zsh
git clone https://github.com/marques576/awsprofile-cli.git && echo 'source "$PWD/awsprofile-cli/awsprofile.sh"' >> ~/.zshrc
```
